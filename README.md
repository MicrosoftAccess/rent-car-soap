
# Rent-car-SOAP

Proyecto rent-car Java 8, maven y jax-ws


## Installation

Install packages and compile first

```bash
  cd mi-soap
  mvn clean install
```

Usage
```bash
  mvn exec:java -Dexec.mainClass="com.rentcar.soap.Main"
```

Client Usage
```bash
  mvn exec:java -Dexec.mainClass="com.rentcar.soap.ClienteSOAP"
```
